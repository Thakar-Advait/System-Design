# System Design & Design Patterns - Code Examples

A comprehensive collection of design pattern implementations and system design examples, written in **Go**. This repository serves as a practical learning resource for understanding software design patterns through hands-on implementation.

This repository exists to:

- **Learn by Doing**: Understand design patterns through real-world implementations rather than just theory
- **Build Practical Skills**: See how design patterns solve actual software engineering problems
- **Practice System Design**: Work through complex problems that combine multiple patterns
- **Reference Guide**: Serve as a code reference for implementing patterns in your own projects

The repository currently focuses on **Creational Design Patterns**, with each pattern demonstrated through multiple practice problems that progressively build understanding. It also includes complex problems that combine multiple patterns to solve real-world scenarios.

### Prerequisites

- **Go 1.19+** installed on your system

### Running Examples

Navigate to any problem directory and run:

```bash
cd creationaldesignpatterns/abstractfactorymethod/pq1
go run main.go
```

Or build and run:

```bash

./pq1  # or the executable name
```
